I build software that performs when the stakes are highest.

- 🏥 Spent 3+ years at **Siemens Healthineers** writing C++ and C# for systems that ship to 1,000+ medical devices in hospitals worldwide — including a remote update platform that saved ~€500K/year.
- ☁️ Currently a **Software Engineer Co-op at Medtronic**, building cloud-native data pipelines on AWS and integrating Visual AI into automated testing.

### 🚀 Featured projects

#### 🧠 [mental-health-signals](https://github.com/punithb19/mental-health-signals) &nbsp;·&nbsp; `Python` `LoRA` `RAG` `FastAPI`
End-to-end ML pipeline that classifies intent + concern severity in support-group posts and generates safe, grounded replies via RAG. Multi-tier crisis detection, FAISS retrieval with cross-encoder re-ranking, RoBERTa fine-tuned with LoRA, and Flan-T5 generation — all behind a FastAPI web UI. 110+ commits and a full test suite.

#### 🗄️ Distributed File Store &nbsp;·&nbsp; `Go` `gRPC` `MinIO` `PostgreSQL` `Docker`
Distributed storage system with secure uploads, versioned retrieval, and metadata management. ~99% data availability via chunk replication, MinIO object storage, and optimized gRPC between Dockerized microservices.

#### 🎨 [artistryhub](https://github.com/punithb19/artistryhub) &nbsp;·&nbsp; `Angular` `Node.js` `MongoDB` `GCP`
Full-stack artist discovery platform powered by the Artsy API. Supports 10,000+ artworks with dynamic search, favorites, and JWT-based auth. Deployed on Google Cloud with MongoDB Atlas.<br>
🔗 [Live demo](https://frontend-dot-artsy2bypunith-429730.wl.r.appspot.com/search)

#### 📱 [artsy-android-app](https://github.com/punithb19/artsy-android-app) &nbsp;·&nbsp; `Kotlin` `Jetpack Compose` `Retrofit`
Native Android client for the Artsy platform — Compose UI, Retrofit + OkHttp networking, Coil image loading, persistent auth, and dark mode.

#### 🤖 [machine-learning-playground](https://github.com/punithb19/machine-learning-playground) &nbsp;·&nbsp; `Jupyter` `Python`
A collection of ML experiments: tree methods on APS failure data, time-series classification on AReM, KNN on vertebral column data, transfer learning for waste image classification (F1 = 0.91), semi-supervised learning, and more.
